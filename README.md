# wx-miniapp
Running platform: 微信小程序
Technology stack: Taro + typescript + redux + taroUi

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
